# Checklist — Sicurezza del laboratorio

Prima di ogni test:

- [ ] Sto usando un ambiente autorizzato.
- [ ] Il target è mio o ho permesso esplicito.
- [ ] Ho definito cosa voglio testare.
- [ ] Ho escluso sistemi reali non autorizzati.
- [ ] Ho salvato snapshot o backup se necessario.
- [ ] So come fermare il test in caso di problemi.
- [ ] Documenterò comandi e risultati.

Durante il test:

- [ ] Non uso comandi distruttivi.
- [ ] Non provo credenziali di altri.
- [ ] Non esfiltrare dati reali.
- [ ] Non pubblico informazioni sensibili.
- [ ] Mi fermo se non sono sicura dell'autorizzazione.

Dopo il test:

- [ ] Salvo le evidenze utili.
- [ ] Scrivo cosa ho imparato.
- [ ] Aggiorno roadmap e note.
- [ ] Rimuovo file temporanei sensibili.
