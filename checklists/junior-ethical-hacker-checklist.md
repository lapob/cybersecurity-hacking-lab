# Checklist — Junior Ethical Hacker

## Fondamenta

- [ ] So usare Linux da terminale.
- [ ] Conosco file, cartelle e permessi.
- [ ] So usare Git e GitHub.
- [ ] Capisco IP, porte, DNS, TCP/UDP.
- [ ] Capisco HTTP, HTTPS, header, cookie e sessioni.

## Metodo etico

- [ ] Chiedo sempre autorizzazione.
- [ ] Definisco lo scope.
- [ ] Uso ambienti di laboratorio.
- [ ] Documento i risultati.
- [ ] Propongo correzioni, non solo problemi.

## Android/Mobile

- [ ] So usare ADB.
- [ ] So leggere Logcat.
- [ ] So installare e rimuovere APK.
- [ ] So leggere AndroidManifest.xml.
- [ ] So usare JADX per analisi base.
- [ ] So usare APKTool per estrarre risorse.
- [ ] So riconoscere permessi rischiosi.
- [ ] So individuare componenti esportati.

## Web

- [ ] So leggere una request HTTP.
- [ ] So leggere una response HTTP.
- [ ] Capisco autenticazione e autorizzazione.
- [ ] Conosco OWASP Top 10.
- [ ] So scrivere note di rischio e remediation.

## Reporting

- [ ] So descrivere una vulnerabilità.
- [ ] So spiegare impatto.
- [ ] So inserire evidenze.
- [ ] So proporre remediation.
- [ ] So scrivere un report leggibile.

## Portfolio

- [ ] Ho appunti ordinati.
- [ ] Ho laboratori documentati.
- [ ] Ho template report.
- [ ] Ho cheatsheet personali.
- [ ] Ho una roadmap aggiornata.
