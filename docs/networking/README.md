# Networking

Questa sezione raccoglie gli appunti sulle reti.

## Fondamenta

- IP address
- Subnet
- Gateway
- DNS
- DHCP
- TCP e UDP
- Porte
- NAT
- Routing

## Tool

- ping
- ip
- netstat / ss
- traceroute
- nslookup / dig
- Wireshark
- nmap

## Comandi rapidi

```bash
ip addr
ip route
ping 8.8.8.8
ping example.com
ss -tulpen
nslookup example.com
traceroute example.com
```
