# Mobile Application Security

Questa sezione è dedicata al pentesting di applicazioni mobile in ambienti autorizzati.

## Argomenti principali

- OWASP Mobile Top 10
- OWASP MASVS
- OWASP MASTG
- Analisi statica
- Analisi dinamica
- Reverse engineering APK
- Traffic interception in laboratorio
- Secure storage
- Authentication e sessioni
- WebView security

## Tool

- ADB
- JADX
- APKTool
- MobSF
- Frida
- Objection
- Burp Suite

## Workflow base

1. Preparare dispositivo/emulatore.
2. Installare APK di test.
3. Raccogliere informazioni con ADB.
4. Analizzare Manifest e codice decompilato.
5. Osservare traffico e log.
6. Documentare risultati e remediation.
